# 🚀 ARAÇ KİRALAMA OTOMASYONU

## 📋 **PROJE HAKKINDA**
Bu proje, **Python** ve **PySide6** kütüphaneleri kullanılarak geliştirilmiş profesyonel bir **Masaüstü Uygulamasıdır.** 

## 🔥 **ANA ÖZELLİKLER**
* **[ARAÇ YÖNETİMİ]** - Yeni araç ekle, sil ve güncelle.
* **[MÜŞTERİ KAYDI]** - Tüm müşteri datalarını güvenle sakla.
* **[KİRALAMA TAKİBİ]** - Hangi araç kimde, anlık görüntüle.
