# 🚗 Araç Kiralama Otomasyonu (Car Rental Automation)

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![PySide6](https://img.shields.io/badge/PySide6-Qt-green?style=for-the-badge&logo=qt)
![MySQL](https://img.shields.io/badge/MySQL-Database-orange?style=for-the-badge&logo=mysql)

Bu proje, araç kiralama şirketleri için geliştirilmiş, kullanıcı dostu bir masaüstü otomasyon sistemidir. Müşteri kaydı, araç yönetimi ve kiralama işlemlerini kolayca ve hızlıca yapmayı sağlar.

## ✨ Özellikler

* **👥 Müşteri Yönetimi:** Sisteme yeni müşteri ekleme ve kimlik/iletişim bilgilerini güvenle saklama.
* **🚘 Araç Kayıt İşlemleri:** Filonuza yeni araçlar ekleme (Plaka, Marka, Model, Günlük Ücret bilgileriyle).
* **📅 Kiralama İşlemleri:** Seçilen tarih aralıklarına göre gün ve toplam tutar hesaplaması yaparak kiralama işlemini tamamlama.
* **📋 Detaylı Listeleme:** Kirada olan araçları, müşteri bilgileri ve kiralama tarihleriyle birlikte tek bir ekranda görüntüleme.

## 🛠️ Kullanılan Teknolojiler

* **Programlama Dili:** Python
* **Arayüz (GUI):** PySide6 (Qt Designer ile tasarlandı)
* **Veritabanı:** MySQL
* **Geliştirme Ortamı:** Visual Studio Code / Qt Creator
